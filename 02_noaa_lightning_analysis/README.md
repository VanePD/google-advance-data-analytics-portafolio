# Project 2: NOAA Lightning Data Analysis

## Objective
This project explores NOAA lightning strike data to understand temporal and spatial patterns using Python and Pandas.

## Dataset
- Source: NOAA (National Oceanic and Atmospheric Administration)
- Description: Lightning strike records including date, time, and geographic information.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Key Steps
- Load and inspect raw lightning data
- Clean and convert datetime fields
- Create temporal features (year, month, day)
- Aggregate lightning counts over time
- Visualize lightning trends
